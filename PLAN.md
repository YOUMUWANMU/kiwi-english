# NZ Year 9 英语生存训练营 · 项目规划

## 项目定位

**不是通用英语学习平台，是"新西兰IEP班专项训练营"。**

目标：孩子9月进IEP班，18个月后去新西兰Year 9，第一天走进教室就能活下去，三个月内融入。

---

## 学情分析

- 英语基础：有一定基础，能简单对话但不流利
- IEP班课程（9月起）：
  - 体育、科学、音乐美术 → 全英文授课
  - 剑桥英语数学 → 每周2节英文课
  - 语文、数学 → 中文授课（有缓冲）
  - 英语教材：新思维英语 + 剑桥主题阅读 + 牛津分级阅读

---

## 平台补位逻辑

学校做的：系统语法、阅读、教材体系
平台做的（学校做不到的）：
1. 标准美式发音强化（口语对话与听力素材均以美式英语为基础）
2. 真实课堂指令听力（"get into pairs" / "hand it in"）
3. 开口说的勇气训练（AI陪练，无压力环境）
4. 学科英语场景（体育课/科学课/音乐课专用词汇）
5. Kiwi文化背景科普（少量NZ本地表达与毛利文化常识，作为文化点缀而非语音主体）

## 18个月路线图

### 第一阶段（月1–6）：生存基础 · 能活下去

**目标：** 6个月末，能走进食堂点餐、问座位、跟陌生同学说"Hi"不卡壳

核心训练：
- 美式发音基础（核心元音辅音、连读弱读规则）
- 课堂生存指令（50个最高频课堂用语）
- 社交开口20句（问候、自我介绍、表达不懂）
- 体育课英语（运动指令、规则说明）
- 学校生活场景（食堂、走廊、厕所、医务室）

### 第二阶段（月7–12）：融入加速 · 能交朋友

**目标：** 12个月末，能读懂老师通知、给老师发邮件、加入一个课外社团

核心训练：
- 科学课英语（实验指令、报告写法）
- 音乐/美术课英语（课堂互动、作品描述）
- 剑桥数学英语（题目读法、解题表达）
- 社交深化（聊兴趣、约朋友、表达喜好）
- 邮件/短信写作（请假、问问题、感谢）
- NZ文化：橄榄球、毛利文化、Kiwi俚语

### 第三阶段（月13–18）：独立自信 · 能发光

**目标：** 出发前，能主动社交、写300字Essay、在课堂上举手发言

核心训练：
- Essay写法（NZ学校标准格式）
- 小组讨论英语（表达意见、同意/反对）
- 演讲/presentation基础
- 深度社交（幽默感、流行语、网络用语）
- 应急情景（生病、被误解、需要帮助）

## 技术架构

### 前端
- 纯HTML + CSS + 原生JS（无框架）
- GitHub Pages托管
- PWA支持（可离线使用基础功能）
- 移动端优先（孩子主要用iPad/手机练习）

### 后端（Cloudflare Workers）
- Claude API代理（口语对话 + 写作批改）
- KV存储（学习进度、生词本、练习记录）
- 内容API（文章、对话、词汇数据）

### 音频策略
- 听力素材：优先使用美式英语TTS（Web Speech API + en-US语音）作为标准发音
- 口语评测：录音 → Claude文字转写 → 发音反馈
- 少量内容（如毛利语问候、NZ本地俚语）标注"NZ文化点缀"，与主体美式发音区分

## 六大模块详细设计

### 模块1：听力训练（Listening）
- 分级美式发音音频（A1→B1）为主，每个阶段穿插1-2个NZ文化点缀片段
- 课堂指令听写练习
- 场景对话：听完选择/填空
- 精听：逐句慢放 + 字幕对照
- 每日"今日一听"（2分钟美式生活片段，定期穿插NZ文化小知识）

### 模块2：口语陪练（Speaking）
- AI角色扮演（老师/同学/店员扮演）
- 场景任务：完成一段对话才算过关
- 发音纠正（提示具体哪个词/音）
- 跟读训练（美式发音示范 → 跟读 → 对比）
- 每周口语挑战（开放话题讨论）

### 模块3：阅读理解（Reading）
- 牛津分级阅读风格（L1→L6）
- NZ主题文章（学校生活、自然、毛利文化）
- 点词释义（点击生词立即显示）
- 阅读理解题（4选1 + 简答）
- 生词本自动收录

### 模块4：写作批改（Writing）
- 引导式写作（给框架/提示词）
- 场景写作：邮件/日记/小作文
- Claude实时批改（语法、用词、表达）
- 错误可视化（标红 + 解释 + 建议）
- 历史作文存档对比进步

### 模块5：学科英语（Subject English）
- 体育课词库 + 对话
- 科学课实验英语
- 数学题目读法
- 音乐/美术课表达
- 每个学科设独立"词卡"

### 模块6：文化通关（Kiwi Culture）
- NZ学校规则（制服、选课、作业）
- 毛利文化基础（Kia ora / Aotearoa）
- NZ流行文化（体育、食物、节假日）
- 社交禁忌与礼仪
- 紧急情况应对

---

## 设计原则

- **语言**：界面中文，练习全英文
- **激励**：每天练习解锁新徽章，18个月有完整成长轨迹
- **无压力**：可以反复练，AI不评判，只给建议
- **家长视角**：有学习报告，家长可查看每周进度
- **内容安全**：所有内容适合12-13岁，无广告
